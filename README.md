# micro-frontends-app
**An initial dive into micro-frontends realm using React microfrontends**


:book: Story: I've always been fascinated about the micrfrontends architecture. This is my first introductory product that will later be improved. If anyone who's interested in micro-frontends and wants to contribute to this project feel free.

## Getting Started
### Open a terminal for each host app and run the commands bellow

 Install dependencies:

        yarn

    or

        npm install

 Run webpack:

        npm start
